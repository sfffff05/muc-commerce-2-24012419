## 用自己的话回答

- 什么是特征，什么是标签：特征（feature）是模型用来做预测的输入变量，比如客户的年龄、消费金额、购买频率等；标签（label）是我们想要预测的目标变量，比如"客户是否会流失"（是/否）。模型通过学习特征与标签之间的关系来完成预测任务。
- 为什么要保留测试集：测试集用于在模型训练完成后评估其性能。如果把所有数据都用于训练，模型可能在训练数据上表现很好，但在真实的新数据上效果很差（过拟合）。保留一部分数据不参与训练，用它来模拟"未见过的数据"，才能客观衡量模型的泛化能力。
- 为什么83%准确率仍可能没有用：准确率不能只看数字本身，还要看数据集的类别分布。假如数据中只有5%的客户流失、95%未流失，一个"全部预测为不流失"的傻瓜模型就能达到95%准确率，而83%反而比它更差。这种情况下需要结合精确率、召回率、F1分数等指标综合评估，尤其是关注少数类别的预测效果。
